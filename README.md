This is a simple image hosting server to allows easy sharing of images with friends

To use please run the  following commands:
pip install -r requirements.txt

This will install the following libarays:

click==8.3.0
Flask==3.1.2
itsdangerous==2.2.0
Jinja2==3.1.6
MarkupSafe==3.0.3
pillow==11.3.0
Werkzeug==3.1.3
